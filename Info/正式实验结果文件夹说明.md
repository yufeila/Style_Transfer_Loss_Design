# 正式实验结果文件夹说明

记录日期：2026-06-20

本文档说明 `Experiments/results/` 下三个正式实验结果文件夹：

- `strong_baseline`
- `color_preservation`
- `semantic_mask_style_transfer_sam_morph`

这三个实验都使用同一组原始图像：

| 项目 | 文件 |
|---|---|
| 内容图 | `/data1/yyf/Style_Transfer_Loss_Design/data/raw/content/golden_gate_bridge.jpg` |
| 风格图 | `/data1/yyf/Style_Transfer_Loss_Design/data/raw/style/composition_vii_kandinsky.jpg` |

本地结果目录位于：

```text
Experiments/results/
```

## 总览

| 文件夹 | 对应实验 | 核心方法 | 训练步数 | 输出重点 |
|---|---|---|---:|---|
| `strong_baseline` | 实验 6.0 | 普通神经风格迁移 | 3000 | 基线结果和训练过程 |
| `color_preservation` | 实验 6.1 | 亮度通道风格迁移 + 内容图颜色保持 | 3000 | 颜色保持结果、亮度/色度参考图 |
| `semantic_mask_style_transfer_sam_morph` | 实验 6.2 | SAM 语义区域 mask + 形态学补全 + masked Gram loss | 3000 | 语义 mask、分区域结果、最终汇总图 |

最终指标摘要：

| 文件夹 | total_loss | content_loss | style_loss | tv_loss |
|---|---:|---:|---:|---:|
| `strong_baseline` | 0.422145 | 0.074446 | 0.000000617 | 0.136019 |
| `color_preservation` | 0.139836 | 0.028337 | 0.000000256 | 0.003647 |
| `semantic_mask_style_transfer_sam_morph` | 0.876101 | 0.183090 | 0.000001047 | 0.128769 |

注意：不同方法的 loss 组成不完全相同，数值主要用于同一实验内部追踪训练过程，不能简单作为跨方法视觉质量的唯一标准。

## 1. `strong_baseline`

目录：

```text
Experiments/results/strong_baseline/
```

### 配置说明

该实验是普通神经风格迁移基线。

主要配置来自 `config.json`：

| 配置项 | 值 | 含义 |
|---|---|---|
| `experiment_name` | `strong_baseline` | 实验名称 |
| `image_size` | `1024` | 输入图像统一缩放到 $1024 \times 1024$ |
| `init` | `noise` | 从随机噪声开始优化生成图 |
| `pooling` | `avg` | VGG19 特征提取使用 average pooling |
| `optimizer` | `lbfgs` | 使用 L-BFGS 优化器 |
| `steps` | `3000` | 优化 3000 步 |
| `learning_rate` | `1.0` | 优化器学习率 |
| `content_layers` | `["conv4_2"]` | 内容损失使用的 VGG 层 |
| `style_layers` | `["conv1_1", "conv2_1", "conv3_1", "conv4_1", "conv5_1"]` | 风格损失使用的 VGG 层 |
| `content_weight` | `3.0` | 内容损失权重 |
| `style_weight` | `300000.0` | 风格损失权重 |
| `tv_weight` | `0.1` | 总变差平滑项权重 |
| `save_every` | `300` | 每 300 步保存一次中间图 |
| `seed` | `42` | 随机种子 |
| `device` | `cuda` | 使用 GPU |

### 文件内容

## 2. `color_preservation`

目录：

```text
Experiments/results/color_preservation/
```

### 配置说明

该实验是颜色保持版本。核心思想是主要优化亮度结构，再结合内容图的颜色信息，使结果更接近原内容图的色彩。

主要配置来自 `config.json`：

| 配置项 | 值 | 含义 |
|---|---|---|
| `experiment_name` | `color_preservation` | 实验名称 |
| `image_size` | `1024` | 输入图像统一缩放到 $1024 \times 1024$ |
| `init` | `noise` | 从随机噪声开始优化 |
| `pooling` | `avg` | VGG19 特征提取使用 average pooling |
| `color_preservation` | `luminance_only` | 使用亮度通道风格迁移方案 |
| `optimize_luminance_channel` | `true` | 主要优化亮度通道 |
| `luminance_histogram_matching` | `true` | 对亮度分布做匹配 |
| `luminance_projection` | `false` | 不启用额外亮度投影 |
| `optimizer` | `lbfgs` | 使用 L-BFGS 优化器 |
| `steps` | `3000` | 优化 3000 步 |
| `learning_rate` | `1.0` | 优化器学习率 |
| `content_layers` | `["conv4_2"]` | 内容损失使用的 VGG 层 |
| `style_layers` | `["conv1_1", "conv2_1", "conv3_1", "conv4_1", "conv5_1"]` | 风格损失使用的 VGG 层 |
| `content_weight` | `3.0` | 内容损失权重 |
| `style_weight` | `200000.0` | 风格损失权重，低于 baseline |
| `tv_weight` | `1.0` | 总变差平滑项权重，高于 baseline |
| `save_every` | `300` | 每 300 步保存一次中间图 |
| `seed` | `42` | 随机种子 |
| `device` | `cuda` | 使用 GPU |

## 3. `semantic_mask_style_transfer_sam_morph`

目录：

```text
Experiments/results/semantic_mask_style_transfer_sam_morph/
```

### 配置说明

该实验是语义区域约束版本。它使用 prompt-guided SAM ViT-H 生成语义 mask，并对桥梁细索区域做形态学补全，然后在语义区域内计算 masked Gram 风格损失。

主要配置来自 `config.json`：

| 配置项 | 值 | 含义 |
|---|---|---|
| `experiment_name` | `semantic_mask_style_transfer_sam_morph` | 实验名称 |
| `image_size` | `1024` | 输入图像和 mask 统一为 $1024 \times 1024$ |
| `init` | `noise` | 从随机噪声开始优化 |
| `pooling` | `avg` | VGG19 特征提取使用 average pooling |
| `optimizer` | `lbfgs` | 使用 L-BFGS 优化器 |
| `steps` | `3000` | 优化 3000 步 |
| `learning_rate` | `1.0` | 优化器学习率 |
| `content_layers` | `["conv4_2"]` | 内容损失使用的 VGG 层 |
| `style_layers` | `["conv1_1", "conv2_1", "conv3_1", "conv4_1", "conv5_1"]` | 风格损失使用的 VGG 层 |
| `content_weight` | `3.0` | 内容损失权重 |
| `style_weight` | `300000.0` | 风格损失权重 |
| `tv_weight` | `0.1` | 总变差平滑项权重 |
| `style_loss_mode` | `masked_gram` | 使用分区域 masked Gram 风格损失 |
| `semantic_masks.method` | `sam_prompt_masks` | 从 SAM prompt mask 文件读取语义区域 |
| `semantic_masks.mask_dir` | `/data1/yyf/Style_Transfer_Loss_Design/masks/sam_bridge_morph_1024` | 远程 mask 来源目录 |
| `semantic_masks.labels` | `sky`, `water`, `bridge_building`, `mountain_land` | 参与语义约束的四类区域 |
| `semantic_masks.region_weights` | 全部为 `1.0` | 四类语义区域等权重 |
| `save_every` | `300` | 每 300 步保存一次中间图 |
| `seed` | `42` | 随机种子 |
| `device` | `cuda` | 使用 GPU |

运行命令记录在 `command.txt`：

```bash
cd /data1/yyf/Style_Transfer_Loss_Design/code
CUDA_VISIBLE_DEVICES=0 python run_style_transfer.py --config configs/semantic_mask_style_transfer_sam_morph.json
```
